The CSS Box Model

What is the CSS Box Model?

Everything in CSS is a box. The CSS Box Model is what wraps each box (HTML element) on a web page - it consists of 4 parts.

  1. Margin - the outer space between the element and other elements
  2. Padding - the inner space between the element content and border
  3. Border - the element border
  4. Content - the element content

Box sizing

The box-sizing CSS property controls how the box model is calculated.

Each box has the same width, heigth, margin, padding and border.

The first box has box-sizing: content-box, the second one box-sizing: border-box

content-box - Total width 216px (padding and border are added on top of width)

border-box - Total width 160px (padding and border are subtracted from content - the box preserves 160px in size)

This eliminates the need to calculate a box's final size.
